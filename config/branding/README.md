# Branding

Coloque nesta pasta a logo usada no cabecalho e na tela de login.

Nome recomendado:

```text
Neovia-Logo-Branco.png
```

Formatos aceitos, nesta ordem de prioridade:

- `Neovia-Logo-Branco.png`
- `Neovia-Logo-Branco.jpg`
- `Neovia-Logo-Branco.jpeg`
- `Neovia-Logo-Branco.svg`

Depois de trocar o arquivo, atualize a pagina no navegador. Se a imagem nao
aparecer de imediato, reinicie o container do sistema.
